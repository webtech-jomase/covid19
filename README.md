# covid19
Todo el contenido dedicado a combatir los efectos del COVID19

21 de Marzo del 2020


Iniciamos este repositorio para aportar de nuestra parte todo lo que esta de nuestra mano que por desgracia es poco ya que nuestra tecnica esta asociada a otros ambitos.

Aun esforzandonos por ser positivos, el panorama actual es desolador: el material medico escasea. Somos conscientes de que existen y existiran otras alternativas: esta es nuestra particular forma de afrontar esta situacion.

La estructura interna de este repositorio constara de una jerarquia en carpetas donde compartiremos los distintos proyectos. El primer nivel de esta jerarquia estara asociado a una problematica y dispondra de un fichero README.md, como este, donde se incluiran los enlaces a distintas plataformas como youtube, thingverse y enlaces para obtener el distinto material.

Dentro de este primer directorio, se iran creando nuevas carpetas precedidas por el prefijo PRJ mas un indice asociado a su precedencia. El indice menor sera siempre el inicial y el mayor el ultimo. Cada PRJ sustituye al anterior y en su README.md se incluira el porque de ser descartado ese enfoque.
